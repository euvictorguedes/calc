## :question: How to use?

### Installation: 
```bash
$ npm install @vitogd/calc
```

### Declaration on javascript:

```js
const calc = require('@vitogd/calc');
```

### Usage:

```js
const sum = calc.sum(2, 2); // 4

const sub = calc.sub(4, 2); // 2

const mult = calc.mult(3, 2); // 6

const div = calc.div(6, 2); // 3
```
